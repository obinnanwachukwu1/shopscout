# Extension Icons

Place your icon files here with the following names:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

For now, you can use placeholder icons or create custom icons with the ShopScout logo (magnifying glass + shopping cart).

## Quick Icon Generation

You can use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Or design in Figma/Canva and export at different sizes

The icons should represent the "ShopScout" brand - a magnifying glass examining products.
